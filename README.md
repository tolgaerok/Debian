## How to run?

1. Make sure `git` is usable<br>
   If not, install it:

```sh
sudo dnf install git -y
```

2. Open Terminal, type:

```sh
git clone tolgaerok/Debian-tolga
cd ./Debian-tolga
```

3. Run it:

```sh
chmod u+x ./install.sh
./install.sh
```
